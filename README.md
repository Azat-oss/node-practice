
# Node.js Color Console App
# Простое консольное приложение на Node.js, демонстрирующее работу с модулями (fs, path, os) и стилизацию вывода с помощью библиотеки chalk.
# Особенности
- Использование пользовательских модулей.
- Работа с системной информацией (ОС, пути, версии Node.js).
- Цветной вывод в консоль с помощью chalk v4.
# Установка и запуск
# Клонирование/Создание папки: 
- Убедитесь, что файлы находятся в структуре:
```text
/project-folder
├── app.js              # Точка входа приложения
├── package.json        # Зависимости и скрипты
├── package-lock.json   # Зафиксированные версии зависимостей
├── README.md           # Этот файл
└── utils
    └── message.js      # Пользовательский модуль с функцией приветствия


# Установка зависимостей:
- Откройте терминал в папке проекта и выполните команду для установки необходимой версии chalk:
- npm install
# Запуск приложения:
- node app.js
- или
- node run start
# Требования
- Node.js (v12 или выше)
